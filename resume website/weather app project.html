JavaScript Weather App

This weather app uses JavaScript and the OpenWeather API to fetch and display real-time weather data based on user input. Users can search by city name and view details like temperature, humidity, and conditions.

I used JavaScript features like DOM manipulation, event handling, and Axios for API requests. This project helped me strengthen my skills in working with APIs.
