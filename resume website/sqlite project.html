SQLite Expense Tracker

This is a simple expense tracking app built in Python using SQLite. It allows users to add, view, and manage their expenses with categories, descriptions, and amounts. Data is stored locally in an SQLite database, making it easy to use offline.

I designed this project to practice database operations like insert, select, update, and delete and allowed users to filter the database by date or category.
