    SQLite Project Weather App Project

JavaScript Weather App

This weather app uses JavaScript and the OpenWeather API to display weather data in real-time. What you do is enter in a city in the place where it says "Enter a city". Then the app uses Axios to send a request to the OpenWeather API and it asks for the current weather data in that city. The user can press whichever data they want to see (E.g. Temperature, humidity, and weatehr conditions like cloudy and sunny). Then the app updates the webpage showing the data the user requested without having to reload the page for the best user experience.
