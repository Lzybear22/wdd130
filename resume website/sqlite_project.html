SQLite Expense Tracker

This project is a really easy to use Expense Tracker made with Python and SQLite. It helps you keep track of your daily spending by alowing users add amounts, categories, and dates for each expense they want to add. All your data is saved locally in a SQLite database, so you don’t need an internet connection or server. You can add, view, update, and delete expenses, and even filter them by date or category. The app also calculates totals to help you see how much you’ve spent. This project helped me learn how to work with databases and use SQL commands inside a Python program.
